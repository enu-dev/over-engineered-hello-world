# 過剰に凝ったHello World

「Hello World」を表示するだけのページに、全力を注いだ結果。

## デモ

[GitHub Pages URL] ← えぬが公開後に記入

## 使い方

ブラウザで `index.html` をダブルクリックするだけ。環境構築不要、サーバー不要。

## 仕掛け一覧

### ローディング画面
偽のブートシーケンス。「フレームワーク: 未インストール（意図的）」など17メッセージが流れ、プログレスバーが進む。

### パーティクルシステム
80個のパーティクルがキャンバスを漂う。Canvas 2D API で描画。
- マウスに近づくと反発する（反発半径: 120px）
- 近い粒子同士はコンステレーション線でつながる（接続閾値: 130px）
- 青色 85% / オレンジ 15% の比率

### 3D チルト
マウス位置に応じてテキストが `perspective(800px) rotateX rotateY` で傾く。

### グリッチエフェクト
3〜8秒のランダム間隔で発火。`text-shadow` の色ずれ + `skewX` を組み合わせた 8 フレームの CSS アニメーション。

### クリックインタラクション

| クリック回数 | 起きること |
|---|---|
| 1〜4回 | リップルエフェクト（波紋） |
| 5回目 | グリッチストーム（6連続グリッチ） |
| 10回目 | 多言語イースターエッグ（9言語でHello World） |
| 以降 5 回ごと | 特大グリッチ |

### コナミコマンド
`↑ ↑ ↓ ↓ ← → ← → B A` → パーティクル爆発 + 10連グリッチ
